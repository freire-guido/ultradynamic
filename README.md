# Ultradynamic Systems

Ultradynamic Systems builds intelligent systems that push people and processes to be smarter, more capable, and more effective. We work across logistics, AI evaluation, education, and AI safety.

## Projects

- **Ultravending**: End-to-end agenting vending machines
- **Ultragaucho**: Powering rioplatense capabilities in LLMs
- **Ultrastudy**: Academic performance prediction markets
- **CtrlShift**: AI policy control platform

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

This site is configured for deployment on [Vercel](https://vercel.com) at [ultradynamic.capital](https://ultradynamic.capital).
